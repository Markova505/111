# Информационная система авторизации пользователей

## Описание
Приложение на C# Windows Forms с системой авторизации, ролями пользователей и интерактивной капчей-пазлом.

## Функционал

### Форма авторизации
- Поля: Логин, Пароль (обязательные)
- Интерактивная капча-пазл (3x3 сетка)
- Блокировка после 3 неудачных попыток (капча или пароль)

### Роли
- **Администратор**: управление пользователями (добавление, редактирование, снятие блокировки)
- **Пользователь**: базовый доступ

### База данных
- SQLite база данных `auth.db`
- Таблица "Пользователи" с полями:
  - Id (PRIMARY KEY)
  - Логин (UNIQUE)
  - Пароль (хешированный BCrypt)
  - Роль (Администратор/Пользователь)
  - Заблокирован (0/1)
  - НеудачныеПопытки

## Запуск
```bash
dotnet restore
dotnet run
```

## Учетная запись по умолчанию
- Логин: `admin`
- Пароль: `admin123`
- Роль: Администратор

## Сообщения системы
- Ошибка входа: "Вы ввели неверный логин или пароль. Пожалуйста проверьте ещё раз введенные данные"
- Успех: "Вы успешно авторизовались"
- Блокировка: "Вы заблокированы. Обратитесь к администратору"
- Дубликат логина: "Пользователь с логином 'X' уже существует"
